# RayswarmAllLibraries

Meta-gem that bundles all Rayswarm library gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'rayswarm-all-libraries'
```

## Included Gems

- **rayswarm-3d** — 3D data visualization engine using Three.js + React + Inertia-Rails
- **biological_it** — Software architecture modeled on biological systems
- **rayswarm-exception** — Exception handling with JSON-RPC-LD support
- **rayswarm-table** — AG Grid table editing engine
