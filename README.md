# EcosystemsAllLibraries

Meta-gem that bundles all ecosystem library gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'ecosystems-all-libraries'
```

## Included Gems

- **service-biological-it** — Software architecture modeled on biological systems
- **service-exception** — Exception handling with JSON-RPC-LD support
- **json-rpc-ld-client** — JSON-RPC-LD client over WebSocket
- **json-rpc-ld-server** — JSON-RPC-LD server
- **rails-multistore-mongodb** — Rails multi-store adapter for MongoDB
- **rails-multistore-couchbase** — Rails multi-store adapter for Couchbase
- **rails-multistore-documentdb** — Rails multi-store adapter for DocumentDB
- **rails-multistore-arangodb** — Rails multi-store adapter for ArangoDB
- **rails-multistore-orientdb** — Rails multi-store adapter for OrientDB
- **rails-multistore-fluree** — Rails multi-store adapter for Fluree
